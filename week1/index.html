Data Limitations

The dataset has missing values in 'Age' (20%) and 'Cabin' (77%). Analysis should consider these gaps when drawing conclusions.

Recommendations
  • Impute missing Age values for better analysis
  • Consider feature engineering (family size, title extraction)
  • Further analyze interaction effects between factors
