7. Key Insights & Conclusion

Executive Summary

The Titanic dataset reveals clear patterns in passenger survival. Through comprehensive analysis of 891 passengers and 12 features, we can identify the most significant factors influencing survival probability.

Key Findings:
  • Data Quality: Age (19.9%) and Cabin (77.1%) have significant missing values
  • Passenger Demographics: Majority were male (64.8%), third class (55.1%), and embarked from Southampton (72.4%)
  • Fare Distribution: Highly skewed with outliers indicating premium tickets
  • Survival Disparities: Clear survival advantages for certain passenger groups

Deployment Instructions

To deploy this dashboard:

  1. Push code to a public GitHub repository
  2. Go to Settings → Pages
  3. Select "main" branch and "/ (root)" folder
  4. Visit your <username>.github.io/<repo-name> URL

Note: This app runs entirely in the browser using JavaScript. No server-side processing is required.

Dataset Adaptation

To use with a different CSV dataset:

  • Update CSV filename in app.js line with file input handling
  • Adjust column names in analysis functions
  • Modify chart configurations for new variables
